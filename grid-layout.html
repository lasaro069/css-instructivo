Grid Layout

Grid Container

Elemento padre principal que va a contener un

        etiqueta{
            display: grid;
        }

y los hijos que van a ser

        etiqueta{
            display:grid-items
        }

Grid Line

Son todas aquellas líneas, tanto horizontales como verticales, internas y externas, que componen un 'grid'

Grid Track

Es el espacio que hay entre dos lineas adyacentes, filas o columnas

Grid Cell

Es el espacio que hay entre dos filas y dos columnas adyacentes

Grid Area

Espacio que rodean 4 'grid lines' sin importar el número de 'grid cell' que contenga



Trabajando con grid container



  • Definimos el contenedor padre y le asignamos una clase
  • definimos los contenedores hijos y les asignamos una clase
  • Definimos valores para los contendeores hijos para poder identificarlos fácilmente
  • le damos las siguientes propiedades y valores al contenedor padre. Esto me divide el contenedor en columnas 
                    .container{
                    display: grid;  /* Damos la caracteristica grid al contenedor*/
                    grid-template-columns: 200px 200px 200px;   /* Definimos la cantidad de comumnas y el tamaño de las mismas*/
                
                }
  • le damos las siguientes propiedades y valores al contenedor padre. Esto me divide el contenedor en filas 
                    .container{
                    display: grid;  /* Damos la caracteristica grid al contenedor*/
                    grid-template-rows: 200px 200px 200px;   /* Definimos la cantidad de filas y el tamaño de las mismas*/
                
                }
  • para reunir el tamaño de filas / columnas en una sola línea utilizamos 
    
                .container{
                    display: grid;  /* Damos la caracteristica grid al contenedor*/
                    grid-template: 100px 200px 300px / 25% 100px 35%; /* Definimos filas / columnas */
                
                }
  • otras opciones de 'display son 
                etiqueta{
                display: subgrid;
                display inline-grid;
            }
    no funcionales por el momento
  • para dar espaciado a las celdas utilizamos la propiedad 'gap' 
                etiqueta{
                column-gap: 5px;   /* asigna espaciado (margen) vertical, entre columnas */
                row-gap: 5px;  /* asigna espaciado (margen) horizontal, entre filas */
                gap: 5px 10px;  /* filas - columnas */
            }
    esta propiedad debe ser aplicada al contenedor padre
  • Para trabajar con 'grid' tenemos una nueva unidad de medida que es la fracción fr
  • la fracción toma el espacio disponible y lo divide en las 'fr' en partes iguales
  • para evitar escribir muchos 'fr ' para dividir columnas o filas podemos utilizar la función repeat() en la propiedad 'grid-template' indicándole en la primera posición el número de filas o columnas y en la segunda posición el tamaño en 'fr' 
              etiqueta{
              display: grid;
              grid-template: 200px 20% 100px / repeat(4, 1fr);
          }
  • además podemos decirle un tamaño min y max en que funcionará 'fr' de acuerdo el tamaño de la pantalla 
            etiqueta{
          display: grid;
          grid-template: 200px 20% 100px / minmax(200px, 1fr)1fr 1fr 1fr;
        }

        etiqueta{
          display: grid;
          grid-template: 200px 20% 100px / repeat(4, minmax(200px, 1fr));
        }
  • para manejar el contenido implícito, es decir, las cajitas sobrantes, utilizamos la propiedad grid-auto-flow: row que es el valor por defecto y los apila en una fila
  • grid-auto-flow: column se apilan ea manera de columna ocupando el espacio estricto
      .contenedor{
        grid-auto-flow: row /* apila los elementos en filas*/
      }

      .contenedor{
        grid-auto-flow: column /* apila los elementos en una columnas*/
      }
      
      .contenedor{
        grid-auto-columns: tamaño de las columnas /* asigna tamaño a las columnas implícitas*/
      }

      .contenedor{
        grid-auto-rows: tamaño filas /* asigna tamaño a las filas de los elementos implícitos*/
      }

Alineacion de filas y columnas

para alineación de columnas (vertical):

Utilizamos la propiedad justify-content con los siguientes valores

  •           .contenedor{
            justify-content: center;  /* centra el contenido en la pantalla*/
          }
        
  •           .contenedor{
            justify-content: start;  /* alinea el contenido al lado izquierdo*/
          }
        
  •           .contenedor{
            justify-content: end;  /* alinea el contenido a la derecha*/
          }
        
  •           .contenedor{
            justify-content: space around;  /* distribuye el contendio de las columnas  con el espaciado indicado. Hay que tener en cuenta que en este caso, el espaciado entre dos items será igual a la suma del espaciado de los items.*/
          }
        
  •           .contenedor{
            justify-content: space-evenly;  /* disptribuye homogeneamente el espacio */
          }
        
  •           .contenedor{
            justify-content: space-between;  /* agrega espacio únicamente entre los elementos */
          }

para alineación de filas (horizontal):

Utilizamos la propiedad align-content con los siguientes valores

  •           .contenedor{
            align-content: space-between;  /* agrega espaciado únicamente entre los elementos */
          }
        
  •           .contenedor{
            align-content: space-evenly;  /* agrega espacio alrededor de los elementos */
          }
        
  •           .contenedor{
            align-content: space-end;  /* alinea el contenido al margen inferior */
          }
        
  •           .contenedor{
            align-content: center;  /* alinea el contenido verticalmente al centro del espacio */
          }



Alineacion de contendio

para alineación del contenido de forma horizontal

Utilizamos la propiedad justify-items con los siguientes valores

  •           .contenedor{
            justify-items: start;  /* coloca el contenido hacia la izquierda, al inicio, solo ocupan el ancho estricot*/
          }
        
  •           .contenedor{
            justify-items: end;  /* alinea el contenido al lado derecho*/
          }
        
  •           .contenedor{
            justify-items: center;  /* alinea el contenido al centro*/
          }
        
  •           .contenedor{
            justify-items: stretch;  /* estira el contenido al espacio asignado por el grid*/
          }

para alineación vertical:

Utilizamos la propiedad align-items con los siguientes valores

  •           .contenedor{
            align-items: start;  /* alinea los elementos al margen superior */
          }
        
  •           .contenedor{
            align-items:end;  /* alinea los elementos a la linea inferior */
          }
        
  •           .contenedor{
            align-items: center;  /* alinea el contenido al centro */
          }
        
  •           .contenedor{
            align-items: stretch;  /* estira los elementos an todo el alto del esapacio del grid */
          }
  • si se quisiera alinear un item de manera independiente, se utilizaría la propiedad 
              .item:nth-of-type(numero del item){
            align-self: ;
            justify-self: ;
          }

Definiendo areas


Para este ejercicio, la primera fila será el 'header', la segunda el 'body' y la tercera el 'footer'

Para esto, utilizamos la propiedad 'grid-template-areas'

            
      .contenedor{
        display: grid;
        grid-template: repeat(3, 1fr) / repeat(4, 1fr);
        gap: 10px;
        height: 100vh;
        grid-template-areas: "header header"  /* nombramos la primera fila y por cada columna escribimos un nombre */
                              "left contenido"
                              "footer footer";
      }

Para los contenedores hijos les damos una clase con los nombres de cada area: 'header', 'left', 'contenido', 'footer' y le coloccamos la porpiedad grid-area

      .header{
        grid-area: header;
      }

Poner nombres a las líneas


Podemos poner nombres a las líneas tanto horizontales como verticales con el fin de poder identificarlas mejor.

Puede dársele cualquier nombre, por ejemplo [inicio], [fin], [inicioBody]

Para esto, utilizamos la propiedad 'grid-template-columns' y 'grid-template-rows'

            
      .contenedor{
        display: grid;
        grid-template-columns: [inicio] 1fr [linea2] 1fr [linea3] 1fr [fin];
        gap: 10px;
        height: 100vh;
      }

      .hijo:nth-of-type(n){
        grid-column: inicio / linea3
      }

Para los contenedores hijos les damos una clase con los nombres de cada area: 'header', 'left', 'contenido', 'footer' y le coloccamos la porpiedad grid-area

      .header{
        grid-area: header;
      }
