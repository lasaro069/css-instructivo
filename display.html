Propiedad 'display'

nos sirve para el contexto de los elementos del navegador. es decir, si tenemos elementos de bloque y de linea podemos modificar su comportamiento

Valores

        {
            display: none /* hace que el elemento no se muestre, pero se sigue cargando, sigue existiendo. */
        }

    
        {
            display: block /* hace que el elemento sea un bloque. Se asigna a elementos de línea */
        }

    
        {
            display: inline /* Hace que el elemento sea de linea. */
        }

    
        {
            display: inline-block /* hace que el elemento sea de línea pero admite medidas y márgenes verticales. */
        }

Propiedad 'text-align'

Es la propiedad que nos permite alinear horizontalmente el contenido de un elemento de bloque siempre y cuando el contenido no tenga un ancho declarado 

        {
            text-align: left /* Alnea el contenido a la izquierda. */
        }
        
    
        {
            text-align: right /* Alinea el contenido a la derecha. */
        }
        
    
        {
            text-align: center /* centra el contenido. */
        }
        
    
        {
            text-align: justify /* alinea el contenido de forma justificada. */
        }
