margen

Elemento de Línea

los elementos de bloque pueden ser dimensionados con 'height' y 'width'

Los elementos de línea no pueden ser dimensionados. el contenido determina el tamaño

'box-sizing: border-box se utiliza para que la medida de la caja sea la medida dada en el heigth y width, manteniendo las prodiedades de border, padding y ajustando el alto y el ancho. esto puede aplicarse a todo el documento con un selector universal

        *{
            box-sizing:border-box;
        }

display: Nos sirve para cambiar el contexto de los elementos del navegador, es decir, si tenemos elementos de bloque y de línea, podemos cambiar su comportamiento con esta propiedad
